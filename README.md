# go-shop

Online Shop with Go Lang + Tailwind CSS

Untuk menjalankan program pertama harus install package yang dibutuhkan dengan perintah:

```
go mod tidy

```

Untuk menjalankan program perintah:

```
go run .

server running di localhost:8080

```
